DATA TYPE IN JS

Two type of data type

  1. Primitive Data type
  2. Non-Primitive Data type

Primitive Data type

  • Number:1,2.....
  • String:'a',"hello"..
  • Boolean:true or false
  • Null:standard value
  • Undefine:let x;
  • Big Int

Non Primitive data type

  • object
  • array
  • function
